To allow this extension downloading pictures from all tabs of current window, open a new tab and type "about:addons" into the address bar, and allow "Access your data for all websites" for this extension.

Unless you do it, the "Download pictures from current window" option simply won't work.


Minimum size of downloaded pictures: 0 px


Maximum size of downloaded pictures: 8000 px

If you're having issues with this extension, report any problems to author, using email from the Mozilla Addons extension page.
